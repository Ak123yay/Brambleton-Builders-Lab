12-session program

Program schedule.

Planned for 6 weeks with 12 total sessions: 11 online sessions plus a final in-person demo day. Online sessions meet Wednesdays and Sundays from 2:00 PM - 4:00 PM.

01

Orientation + HTML foundations

Students set up tools and build a first About Me webpage.

02

HTML build lab

Students customize structure, links, images, buttons, and sections.

03

CSS styling foundations

Students turn plain pages into styled card-style webpages.

04

Layout + design lab

Students add navigation, cards, spacing, and page sections.

05

JavaScript basics

Students add button clicks, functions, dynamic text, and simple conditions.

06

Interactive feature lab

Students build a quiz, counter, quote generator, theme toggle, or study helper.

07

Responsible AI + debugging

Students learn safe AI use, test suggestions, and fix broken code.

08

Final project planning

Students choose a project, audience, features, and wireframe.

09

Build sprint 1

Students build final project structure, content, and styling.

10

Build sprint 2 + polish

Students complete one JavaScript feature and improve layout quality.

11

Demo rehearsal + final fixes

Students practice a one-minute demo and finish final debugging.

12

In-person demo day

Students present final projects to peers, parents, and mentors.

Final demo

Build it. Test it. Explain it.

The goal is not just working code. Students should understand what they made.

Responsible AI rules

AI is a helper, not a shortcut.

Use AI to explain errors.

Students ask for beginner-friendly explanations, then apply the fix themselves.

Do not paste private information.

No private student, parent, account, or contact details should go into AI tools.

Do not submit code you cannot explain.

Every final project should be explained in plain English during the demo.

Test AI-generated code.

Suggestions only count after they work in the browser and the student understands them.
