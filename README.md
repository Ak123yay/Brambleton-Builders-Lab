# Brambleton Builders Lab

Brambleton Builders Lab is a planned Summer 2026 beginner coding and responsible AI program for rising 6th-9th grade students.

Learn to code. Build real projects. Use AI responsibly.

The program is designed as a free 6-week online program with guided build-alongs, debugging support, responsible AI mini-lessons, and a final showcase-ready project.

## Program Highlights

- 100% free
- Beginner-friendly
- No previous coding experience needed
- For rising 6th-9th grade students
- Live online lessons twice a week
- Guided build-alongs and debugging help
- Responsible AI safety and usage lessons
- Final demo day / showcase project

## Planned Schedule

- Days: Wednesdays and Sundays
- Time: 2:00 PM - 4:00 PM
- Length: 6 weeks
- Format: 12 total sessions
- Online sessions: 11 lessons/labs
- Final event: planned in-person showcase / demo day

## What Students Will Learn

- HTML and CSS foundations
- Basic JavaScript interaction
- Debugging and problem-solving
- Responsible AI use
- How to present a final project clearly

## What Students Will Build

Students will work toward a final project they can show and explain. Example project paths include:

- Personal website
- Club or nonprofit page
- Study helper web app
- Interactive JavaScript project
- Final showcase-ready project

## Responsible AI Focus

Students should learn to use AI as a helper, not a shortcut. The program emphasizes safe, thoughtful, and testable AI use:

- Do not paste private information into AI tools.
- Use AI to understand errors and debugging steps.
- Test AI-generated code before trusting it.
- Be able to explain any code used in the final project.

## Local Development

This is a static website. No build step is required.

Open `index.html` in a browser, or run a simple local server from this folder:

```powershell
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
